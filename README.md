
# Gas Detector Project

This repository contains a past project: a gas detection system built with an ESP32, MQ-5 sensor, buzzer, and Blynk integration.

## Overview
 Detects flammable gases and triggers alerts.
 Specifically Methane, Carbon Monoxide and other dangerous gasses.
 Provides live monitoring and manual buzzer control via the Blynk app.
 Focused on safety and real-time responsiveness.
 For full documentation please refer to the docs and see report and research plan.
